Ride the Lightning

Today's digital world has brought us fantastic new technologies one of the most fasinating being cryptocurrencies. Using advanced hashing algorithms cryptocurrencies like Bitcoin, Dogecoin, and Litecoin etc.. create a decentralized way and very inexpensive way of sending a form of currency through the internet. We think that is awesome here at Cryptallica. With so many cryptocurrencies making a great (and sometimes negative) impact on our world it can be hard to keep up or even start up. This is why we aim to create software that helps using, developing, and creating crptocurrency platforms.

What do we do?

Cryptocurrency software with a good dose of metal. Currently we offer a multi cryptocurrency wallet, a cryptocurrency donator, and coming soon a cryptocurrency cart manager. All of our software is open-source free software released under GPL v2.0 and always remain so. Feel free to download, hack, or contribute to the Cryptallica project. We are run purely on donations and drive for great applications. View our software below.

Cryptallica Wallet

Main Page Github Download Documentation

The Cryptallica wallet is a multi cryptocurrency wallet complete with all essential function found in a cryptocurrency wallet such as sending coins, encrypting your wallet, importing addresses and keys, and backing up your wallet. The wallet works with a number of coins listed below. Along with typical coin functions, the wallet also offers a QR camera and file reader. Our wallet is programmed with security and freedom in mind and all of our code is hosted on github and the cryptallica site. Like all of Cryptallica's products, our wallet is free under a GPL liscence.

Check it out

The Cryptallica Donator is an easy to use JavaScript cryptocurrency address donator widget. The donator is pure javascript and uses the canvas to create dynamic QR codes, but the donator also offers a fallback for using images as QR codes. Setting is up is very easy and the package is only 100kb. Above is also all of the current built in coins. Adding coins is also very easy. Vist the doc.
